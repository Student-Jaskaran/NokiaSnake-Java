# 🐍 Nokia Snake – Java

A simple Java version of the iconic Nokia Snake game — eat, grow, and try not to crash!

## 🎮 Features
- Classic snake movement
- Food generation and scoring
- Game over detection

## 🛠 Tech Stack
- Java
- Java Swing

## 🚀 Run
Open the project in IntelliJ and run `SnakeGame.java`.
